# Color

Color is really very important in any UI framework.

In Jetpack Compose we have various way to use color.a

## Built in colors

| Code              | Hex-Code   |          Color           |
|-------------------|------------|--------------------------|
| Color.Black       | 0xFF000000 | <Color color="#000000"/> |
| Color.DarkGray    | 0xFF444444 | <Color color="#444444"/> |
| Color.Gray        | 0xFF888888 | <Color color="#888888"/> |
| Color.LightGray   | 0xFFCCCCCC | <Color color="#CCCCCC"/> |
| Color.White       | 0xFFFFFFFF | <Color color="#FFFFFF"/> |
| Color.Red         | 0xFFFF0000 | <Color color="#FF0000"/> |
| Color.Green       | 0xFF00FF00 | <Color color="#00FF00"/> |
| Color.Blue        | 0xFF0000FF | <Color color="#0000FF"/> |
| Color.Yellow      | 0xFFFFFF00 | <Color color="#FFFF00"/> |
| Color.Cyan        | 0xFF00FFFF | <Color color="#00FFFF"/> |
| Color.Magenta     | 0xFFFF00FF | <Color color="#FF00FF"/> |
| Color.Transparent | 0x00000000 | <Color color="#00000000"/> |

## Hex code

We can use hex code as well for any custom color.

```kotlin
Color(0xff00ffa2)//100% opacity
```

<Color color="#00ffa2" width="180px" height="80px" margin="24px"/>


```kotlin
Color(0x3200ffa2)//50% opacity
```

## RGB

```kotlin
Color(red = 255, green = 0, blue = 0, alpha = 255)
```

<Color color="#ff0000ff" width="180px" height="80px" margin="24px"/>

## HSL

```kotlin
Color.hsl(hue = 50f, saturation = 0.5f, lightness = 0.5f, alpha = 1f)
```

<Color color="hsl(50, 50%, 50%)" width="180px" height="80px" margin="24px"/>

## HSV

```kotlin
Color.hsl(hue = 50f, saturation = 0.5f, lightness = 0.5f, alpha = 1f)
```

<Color color="hsl(50, 50%, 50%)" width="180px" height="80px" margin="24px"/>

## Alpha/Opacity

### Alpha by hex code

If you are using `hex code` then control opacity directly via the first two hex digit.

Color(0x`32`00ffa2)// here 32 is for 50% alpha

<Color color="#00ffa232" width="180px" height="80px" margin="24px"/>

<DecToHex value="100"></DecToHex>

### Alpha by copy extension

You can set `alpha` by the following way as well:

```kotlin
Color.Red.copy(alpha = 0.5f)
// or
val myColor = Color.Red
myColor.copy(alpha = 0.5f)
```

<Color color="#ff000032"/>

This way you can put value in `0 to 1` range. For `50%` alpha you have to put `0.5`.
