# Text

Text is a central piece of any UI, and Jetpack Compose makes it easier to display or write text.

## How to show a text?

If we want to show **Hello World**, then we can use the following code:

```kotlin
Text("Hello World")
```

<Image src="/img/text.png" alt="Text"/>

----------------------------------------------------

## Color

We can use the `color` parameter to specify the color

```kotlin
Text(
    "Hello World",
    color = Color.Red
)
```

<Image src="/img/red-text.png" alt="Red Text"/>

----------------------------------------------------

## Font size

Use `fontSize` parameter to specify the font-size

```kotlin
Text(
    "Hello World",
    fontSize = 20.sp
)
```

<Image src="/img/text-font-size.png" alt="Font Size of text"/>

----------------------------------------------------

## Font weight: bold, normal

Use the `fontWeight` parameter to specify the font-weight(bold, normal, etc.)

```kotlin
Text(
    "Hello World",
    fontWeight = FontWeight.Bold
)
```

<Image src="/img/font-weight.png" alt="Font weight of text"/>

Few useful available weights are:

* Thin
* Normal
* Medium
* Bold, etc.

----------------------------------------------------

## Font style: italic, underline, strike-through

Use `fontStyle` parameter to have italic, underline, strike-through text

```kotlin
Text(
    "Hello World",
    fontStyle = FontStyle.Italic
)
```

<Image src="/img/text-font-style.png" alt="img.png"/>

Other styles are:

* Italic
* Underline
* StrikeThrough

----------------------------------------------------

## Font family

Use `fontFamily` param to specify font-family.

```kotlin
Text(
    "Hello World",
    fontFamily = FontFamily.Cursive
)
```

<Image src="/img/font-family.png" alt="img.png"/>

Default fonts are:

* Cursive
* Default
* Serif
* Monospace
* SansSerif

----------------------------------------------------

## Letter spacing

Use `letterSpacing` parameter to mention letter-spacing.

```kotlin
Text(
    "Hello World",
    letterSpacing = 5.sp
)
```

<Image src="/img/letter-spacing.png" alt="Letter spacing"/>

----------------------------------------------------

## Text align

Use `textAlign` to align text.

```kotlin
Text(
    "Hello World, again and again and again and again and again and again",
    textAlign = TextAlign.Center
)
```

<Image src="/img/text-align.png" alt="img.png"/>

----------------------------------------------------

## Line height

Use `lineHeight` to adjust the line-height.

```kotlin
Text(
    "Hello World, again and again and again and again and again and again",
    lineHeight = 50.sp
)
```

<Image src="/img/line-height.png" alt="Line height"/>

----------------------------------------------------

## Max lines

Use `maxLines` to limit the number of lines the will span.

```kotlin
Text(
    "Hello World, again and again and again and again and again and again then end",
    maxLines = 1
)
```

<Image src="/img/max-lines.png" alt="Max lines"/>

----------------------------------------------------

## Overflow

Use `overflow` parameter to handle overflow situation.

```kotlin
Text(
    "Hello World, again and again and again and again and again and again then end",
    maxLines = 1,
    overflow = TextOverflow.Ellipsis
)
```

<Image src="/img/overflow.png" alt="Text Overflow"/>
